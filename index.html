SEM暑培

清华大学经管学院第一期计算机技能暑期培训
Tip

欢迎来到 SEM Coding Summer 2025！

本项目由清华大学经济管理学院科协主办，与计算机系合作，是面向经管学院同学开设的系统性计算机科学与编程技能培训。

在人工智能技术与社会各领域深度融合的背景下，计算思维与编程技能已成为文科研究与职业发展的关键能力。本培训旨在帮助非计算机专业的同学理解计算机科学的核心逻辑，掌握现代编程工具，并能将其应用于各自的学科领域，从而在人工智能时代获得独特的交叉学科优势。

我们相信，代码是赋能思想的工具。本课程将引导你从零开始，构建坚实的计算机知识体系。

课程目标与内容

本培训课程体系完整，分为三大模块，由浅入深，旨在帮助学员建立全面的知识结构。

模块一：计算机科学基础

本模块致力于为无基础的学员构建扎实的理论与编程基础，清晰地阐释计算机科学的基本范式。

  • 计算机科学与程序设计语言导论：讲解计算机系统的基本工作原理，分析不同编程语言（如 Python 与 C++）的设计哲学及核心应用领域。
  • Python 语言入门：系统讲授 Python 的核心语法，包括变量、数据结构、流程控制、函数，并介绍面向对象编程（OOP）的基本思想。
  • Python 实用进阶：重点培养在命令行环境下进行程序开发与调试的能力，讲解参数处理及使用 pdb 进行代码调试的方法。

模块二：核心编程应用实践

本模块聚焦于将编程技能应用于解决实际问题，重点培养学员的数据获取、处理与分析能力。

  • 版本控制与项目协作：讲授 Git 与 GitHub 的使用，使其掌握现代软件开发中的标准化项目管理与协作流程。
  • 网络数据采集：学习 Python 爬虫 的原理与实践，掌握从静态及动态网站自动获取公开数据的技术。
  • 数据分析与可视化：介绍如何使用 Matplotlib 和 Seaborn 等库进行数据可视化，将数据以直观的图形方式呈现。
  • 科学计算：讲授如何运用 NumPy 和 SciPy 等库进行高效的科学计算，为学术研究中的数据处理提供支持。
  • 开源项目复线与应用：带领同学复现并且使用别人的大规模项目。
  • 基于大模型的自动化流程构建：探索基于大型语言模型的自动化流程，体验人工智能在提升工作流效率方面的应用。

模块三：计算思维与前沿展望

本模块旨在拓宽学员的学术视野，展示计算思维在不同前沿领域的交叉应用。

  • 量化金融：介绍金融市场数据分析、投资组合构建与风险管理的基本原理，学习如何使用Python实现基础的金融模型，包括时间序列分析、资产定价模型及回测框架，帮助同学将编程技能应用于金融决策与研究。
  • 大型语言模型推理：探索大模型的形式化和非形式化推理能力，深入理解DeepSeek-R1等前沿模型在复杂推理任务中的应用，学习如何设计有效的提示以激发模型的推理潜力，并探讨大模型在科研辅助、逻辑分析和决策支持方面的创新应用。

教学方法

  • 面向非计算机背景学生：本课程内容的设计充分考虑了无编程基础学生的学习路径，从基本概念出发，避免了不必要的复杂性。
  • 理论与实践结合：课程遵循“实例驱动”的教学模式，每个知识点都配有具体的代码示例和实践任务，确保学员在动手操作中巩固理解。
  • 工具驱动：课程的核心目标是让学员掌握能够解决实际问题的工具。例如，你将学习如何运用 pandas 等库高效地完成数据分析任务，或利用 transformers 库调用前沿的人工智能模型。

适合对象

  • 清华大学所有对计算机科学、人工智能和数据分析感兴趣的文科院系学生（包括但不限于经管、人文、社科、法律、新闻、艺术等专业）。
  • 希望系统性提升数据处理与分析能力，为学术研究或未来职业发展构建核心技能的同学。
  • 无任何编程基础，但希望开始学习编程的同学。
Tip

本课程不要求任何计算机科学的先修知识。
